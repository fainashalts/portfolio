Educational website discovery application geared toward individuals learning to program.

Essentially a "reddit for programmers." Users can peruse various programming topics, view relevant links, upvote them, and add their own links as well as comment on existing links. Users can also view links they have upvoted on their profile page.

  • Ruby/Ruby on Rails
  • PostgreSQL database with Active Record
  • Bootstrap/SASS for responsive design
  • Link Thumbnailer Gem to upload link information
  • Acts_As_Votable gem to allow for upvotes and linking of upvoted links to users
  • Devise gem for user authentication
  • JQuery
  • Serves API for topics, subtopics, links, and users
  • Testing with RSpec
  • Deployed on Heroku

Ridesharing coordination application to help users find others who want to commute together to and from the same places at the same time.

Users can search for existing carpools, within a five-mile radius of their searched origin and destination addresses. Users can then join a carpool to be added to that carpool's coordination page. Users may also create their own carpool group if they are dissatisfied with the results of their search.

  • AngularJS as a front-end framework, providing client-side interface and consuming API from the back-end
  • Ruby on Rails to provide backend support through a users and carpools API
  • PostgreSQL database with Active Record
  • Bootstrap/SASS for responsive design
  • Geocoder gem to geocode addresses provided by users into latitude and longitude, as well as to provide the ability to query backend API based on locations near user search terms
